О проекте

🌐 Что такое CryptoStart AI?

CryptoStart AI — это образовательный проект о криптовалютах, созданный для изучения блокчейна, цифровых активов и современных AI-технологий.

🎯 Цель проекта

Помочь новичкам разобраться в криптовалютах, понять основы безопасности и научиться принимать осознанные инвестиционные решения.

⚙ Технологии

HTML • CSS • Vercel • Serverless API • Groq AI • CoinGecko API

🔐 Безопасность

Мы обучаем принципам безопасного хранения криптовалют, использованию холодных кошельков и двухфакторной аутентификации.
