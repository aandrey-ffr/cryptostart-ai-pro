О проекте

Что такое CryptoStart AI?

CryptoStart AI — это образовательный проект о криптовалютах, созданный для изучения блокчейна, цифровых активов и современных AI-технологий.

Цель проекта

Помочь новичкам разобраться в мире криптовалют, понять основы безопасности и научиться принимать осознанные решения.

Технологии

• HTML / CSS • Vercel • Serverless API • AI интеграция (Groq) • Работа с внешними API (CoinGecko)
